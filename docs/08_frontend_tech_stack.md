# フロントエンド技術スタック

## スタイリング系

| 分類                        | ライブラリ       | 理由                                                      |
| :-------------------------- | :--------------- | :-------------------------------------------------------- |
| CSS フレームワーク          | **Tailwind CSS** | コンポーネント設計しやすく、TanStack Start とも相性抜群。 |
| UI コンポーネントライブラリ | **shadcn/ui**    | Tailwind ベースで、柔軟・拡張性のある UI 作成ができる。   |

## フォーム・バリデーション系

| 分類           | ライブラリ  | 理由                                                               |
| :------------- | :---------- | :----------------------------------------------------------------- |
| フォーム管理   | **conform** | 型安全なフォーム設計ができ、バリデーションとの親和性も高い。       |
| バリデーション | **ArkType** | Zod よりも柔軟かつ高速。型定義とバリデーションを同時に管理できる。 |

## 認証系

| 分類 | ライブラリ      | 理由                                          |
| :--- | :-------------- | :-------------------------------------------- |
| 認証 | **supabase-js** | Supabase 公式クライアント。認証機能をカバー。 |

## API 通信・クエリ管理系

| 分類                       | ライブラリ                    | 理由                                             |
| :------------------------- | :---------------------------- | :----------------------------------------------- |
| データ取得・キャッシュ管理 | **TanStack Query** (標準搭載) | フェッチ・キャッシュ・再取得・エラー処理を包括。 |

## 状態管理系

| 分類               | ライブラリ                        | 理由                                         |
| :----------------- | :-------------------------------- | :------------------------------------------- |
| グローバル状態管理 | **TanStack Store** (必要に応じて) | 小規模　中規模の状態管理に最適。Redux 不要。 |

## ユーティリティ系

| 分類           | ライブラリ                  | 理由                                                   |
| :------------- | :-------------------------- | :----------------------------------------------------- |
| 日付操作       | **Temporal API** (Web 標準) | Web 標準。最新ブラウザサポート進行中。ライブラリ不要。 |
| アイコンセット | **lucide-react**            | 軽量・モダンなアイコンパック。shadcn/ui 推奨。         |

## フレームワーク

- TanStack Start
- TanStack Query
- TanStack Router
